## G.E.A.R - generic electronics augmentation rig - tools

### Installation
    $ mkdir -p $HOME/.config/gear
    $ cd $HOME/.config/gear
    $ git clone https://github.com/hakt0r/gear modules
    $ cd modules
    $ make install

### CLI usage
    $ gear help
